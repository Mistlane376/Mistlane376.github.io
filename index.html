我的第一篇文章
发表于2026-04-11|日常
写在前面欢迎来到临涧路的精神小屋。 建立这个博客的念头由来已久。作为一个计算机专业的学生，在信息爆炸的时代里，我时常感到被海量的输入淹没，而缺少系统性的输出——没有整理的笔记、没有沉淀的思考、没有被记录下来的成长轨迹。 于是就有了这里。 希望这个地方成为我的精神自留地，记录学习的过程、生活里的灵感，以及那些如果不马上写下来就会消散的想法。如果恰好对正在阅读的你也有帮助，那便是额外的幸运。 这个博客会写什么目前规划的几个方向： 技术笔记：编程语言、框架工具、踩坑记录 项目复盘：课程设计、个人项目的总结 大学生活：日常随笔，成长中的困惑与收获 阅读存档：书摘与读后感 不拘泥于形式，保持更新本身就是胜利。 关于我计算机科学与技术大一学生。喜欢简单优雅的代码，相信文字的力量。 目前使用 Hexo + Butterfly 搭建本站，部署在 GitHub Pages。博文使用 Markdown 撰写——简单、专注、可移植性好。 道阻且长，行则将至。 感谢你的到来。
