- 光泽：从左到右扫过（0.5s）

## 📝 迁移指南

### 从 v3.0.0 升级

无需特殊迁移步骤，只需：

1. **拉取最新代码**
   ```bash
   git pull origin source
   ```

2. **安装依赖**
   ```bash
   npm install
   ```

3. **清理缓存**
   ```bash
   hexo clean
   ```

4. **重新生成**
   ```bash
   hexo g
   ```

5. **本地预览**
   ```bash
   hexo s
   ```

6. **部署到线上**
   ```bash
   hexo d
   ```
