## 代码块阴影效果测试

这里是一段测试代码，用来验证代码块阴影是否占满整个代码块区域：

```javascript
// 这是一个测试代码块
function testCodeBlock() {
  const message = "Hello, World!";
  console.log(message);
  
  // 测试阴影效果
  const shadow = {
    offsetX: 0,
    offsetY: 6,
    blur: 24,
    color: "rgba(0, 0, 0, 0.12)"
  };
  
  return shadow;
}

testCodeBlock();
```

## Python 代码测试

```python
def fibonacci(n):
    """计算斐波那契数列"""
    if n <= 1:
        return n
    else:
        return fibonacci(n-1) + fibonacci(n-2)

# 测试代码块样式
for i in range(10):
    print(f"F({i}) = {fibonacci(i)}")
```

## 长代码测试

```java
public class CodeBlockTest {
    private String name;
    private int value;
    
    public CodeBlockTest(String name, int value) {
        this.name = name;
        this.value = value;
    }
    
    public void displayInfo() {
        System.out.println("Name: " + name);
        System.out.println("Value: " + value);
        System.out.println("Shadow should cover the entire code block area!");
    }
    
    public static void main(String[] args) {
        CodeBlockTest test = new CodeBlockTest("Shadow Test", 100);
        test.displayInfo();
    }
}
```

## 总结

✅ 代码块阴影应该：
- 占满整个代码块区域
- 包括行号区域
- 悬停时阴影加深
- 视觉效果统一美观
